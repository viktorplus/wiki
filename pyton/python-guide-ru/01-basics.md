# 01 • Основы языка

## «Привет, мир!»
```python
print("Привет, Python!")
```

Запуск в PowerShell:
```powershell
python hello.py
```

## Режимы запуска
- **Скрипт**: `python script.py`
- **Интерактивный репл**: `python` или `python -i script.py`
- **Из модуля**: `python -m module.name`
- **Shebang** (Unix): `#!/usr/bin/env python3`

## Модули и пакеты
- Повторно используемый код складывайте в `.py`-файлы и импортируйте с помощью `import module` или `from module import symbol`.
- Для превращения папки в пакет добавьте `__init__.py` (может быть пустым).
- В больших проектах предпочтительны абсолютные импорты.

## Стандартные типы данных
| Тип | Пример | Комментарий |
| --- | --- | --- |
| Список | `[1, 2, 3]` | изменяемый, сохраняет порядок |
| Кортеж | `(1, 2, 3)` | неизменяемый, сохраняет порядок |
| Словарь | `{'ключ': значение}` | отображение по хэшу |
| Множество | `{1, 2}` | уникальные элементы |
| `namedtuple` / `dataclass` | `from dataclasses import dataclass` | лёгкие «записи» |

## Управление потоком
```python
for item in sequence:
    if item % 2 == 0:
        continue
    else:
        break
else:
    print("цикл завершился")
```

## Функции
- Определяйте функции как `def func(arg: type) -> return_type:` с понятной докстрокой.
- Старайтесь использовать ключевые аргументы (keyword-only) для новых интерфейсов.

## Аннотации типов
```python
def square(n: int) -> int:
    return n * n
```
- Для сложных структур используйте типы из `typing` (`List`, `Dict`, `Union`, `Protocol`).

## Упаковка и публикация
- Описывайте проект в `pyproject.toml` или `setup.cfg`.
- Устанавливайте локально `pip install .` или `pip install -e .`.
- Собирайте дистрибутив `python -m build` (пакет `build`).
