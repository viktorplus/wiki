# Справочник по Python

Эта папка содержит краткие инструкции, примеры и команды, помогающие работать с Python. Структура повторяет справочник по Linux, но ориентирована на язык, инструменты и актуальные сценарии разработки на Python.

## Содержание
- [01-basics.md](01-basics.md) — основы языка, запуск и упаковка модулей
- [02-environment.md](02-environment.md) — окружения, менеджмент версий, файлы зависимостей
- [03-tools.md](03-tools.md) — редакторы, линтеры, тесты и дебаг
- [04-workflows.md](04-workflows.md) — повседневные паттерны, CI и измерения производительности
- [05-resources.md](05-resources.md) — полезные ссылки, книги и обучающие траектории

## Примеры
- `examples/` — рабочие скрипты и пакет: `hello.py`, `run_demo.py`, а также `examples/package` с функцией `greet`.

Следуйте этому шаблону, чтобы поддерживать консистентность справочника.
