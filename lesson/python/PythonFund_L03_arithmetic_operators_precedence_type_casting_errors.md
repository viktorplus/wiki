## 2) Арифметические операции
| Оператор | Название | Что делает |
|---|---|---|
| `+` | сложение | складывает значения |
| `-` | вычитание | вычитает одно из другого |
| `*` | умножение | умножает значения |
| `/` | деление | **всегда** даёт `float` результат |
| `//` | целочисленное деление | берёт целую часть, **округляет вниз** |
| `%` | остаток от деления | возвращает остаток |
| `**` | степень | возводит в степень | fileciteturn36file0 fileciteturn36file1

Важно:
- при обычном делении `/` результат всегда дробный (`float`);
- если хотя бы один операнд `float`, результат обычно `float`;
- `//` округляет **в меньшую сторону** (floor). 

### Примеры
```py
print(3 + 5)     # 8
print(7 - 2)     # 5
print(4 * 6.0)   # 24.0
print(8 / 2)     # 4.0
print(7 // 2)    # 3
print(7 % 3)     # 1
print(2 ** 3)    # 8
```

## 3) Переменные и “одноразовые” вычисления
- Если результат понадобится повторно — сохраняем в переменную.
- Если действие делается один раз — хранить в переменной не обязательно. 

```py
sum1 = 3 + 5
a = sum1 + 1
b = sum1 * 3
print(a, b)
```

## 4) Ошибка `ZeroDivisionError`
`ZeroDivisionError` возникает при:
- делении на ноль (`/` и `//`);
- остатке от деления на ноль (`%`).

```py
a = 10
b = 0
print(a / b)   # ZeroDivisionError
```

## 5) Приоритет операций (как в математике)
Порядок выполнения (от высокого к низкому):
1. `()` скобки
2. `**` степень
3. `*`, `/`, `//`, `%` (слева направо)
4. `+`, `-` (слева направо) 

Примеры:
```py
print((2 + 3) * 4)   # 20
print(2 * 3 ** 2)    # 18, т.к. 2 * (3 ** 2)
print(10 / 2 * 3)    # 15.0, слева направо: (10/2)*3
print(10 - 2 + 5)    # 13, слева направо: (10-2)+5
```

## 6) Неизменяемые (immutable) типы данных — идея
В Python есть:
- **неизменяемые** типы (immutable) — нельзя “поменять объект”, можно только создать новый и перепривязать переменную;
- **изменяемые** (mutable) — можно менять содержимое объекта. 

Пример идеи (immutable на числе):
```py
num = 5
num + 2
print(num)      # 5 (ничего не изменилось)

num = num + 2
print(num)      # 7 (переприсвоили)
```

## 7) Операторы приращения (сокращённое присваивание)
| Оператор | Эквивалент |
|---|---|
| `a += 1` | `a = a + 1` |
| `a -= 1` | `a = a - 1` |
| `a *= 2` | `a = a * 2` |
| `a /= 2` | `a = a / 2` |
| `a //= 2` | `a = a // 2` |
| `a %= 3` | `a = a % 3` |
| `a **= 2` | `a = a ** 2` | fileciteturn36file0 fileciteturn36file1

Примеры:
```py
a = 5
a += 1
print(a)  # 6

a = 5
a *= 2
print(a)  # 10
```

## 8) Множественное присваивание
### Несколько значений за раз
```py
a, b, c = 1, 2, 3
print(a, b, c)  # 1 2 3
```

### Одно значение — нескольким переменным
```py
a = b = c = 0
print(a, b, c)  # 0 0 0
```

### Обмен значениями без временной переменной
```py
a, b = 5, 10
a, b = b, a
print(a, b)  # 10 5
```

## 9) Преобразование типов (Type Casting)
Преобразование типов нужно, когда данные разных типов нельзя напрямую использовать в операции  
(например, деление строк вызовет `TypeError`).

### Неявное преобразование (implicit)
Python иногда сам приводит типы, например `int -> float`:
```py
result = 5
result += 3.14
print(type(result))  # float
```

### Явное преобразование (explicit)
Встроенные функции:
- `int("10")`
- `float("10.5")`
- `str(10)`
- `bool(1)` 

Примеры:
```py
a = 5
b = str(a)     # "5"

a = "10"
b = int(a)     # 10

a = 10.7
b = int(a)     # 10 (дробная часть отбрасывается)
```

## 10) Ошибка `ValueError`
`ValueError` возникает, когда тип аргумента допустим, но **значение** некорректно (например, строка не число).

```py
a = "abc"
b = float(a)   # ValueError
```

### 12.3 Остаток и целочисленное деление без `%` и `//`
Идея:
- целая часть: `q = int(a / b)` (для положительных чисел как в примерах)
- остаток: `r = a - q*b`
```py
a = int(input("Введите первое число: "))
b = int(input("Введите второе число: "))

q = int(a / b)
r = a - q * b

print("Остаток от деления:", r)
print("Целочисленное деление:", q)
```
