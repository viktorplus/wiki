# Python Fundamentals — Lesson 7: цикл `while`, `break`, `continue`, `while/else`, модуль `random`

---

## 0) Термины
- **Цикл** — повторяет выполнение блока кода несколько раз, пока выполняется условие.
- **Итерация** — одно выполнение тела цикла.
- **`while`** — цикл **с предусловием**: перед каждой итерацией проверяется условие. 

---

## 1) Синтаксис `while`
```py
while condition:
    # тело цикла
    # (обычно внутри меняем переменные,
    #  чтобы когда-то выйти из цикла)
```
- Условие проверяется **перед** каждой итерацией.  
- Как только условие становится `False` → цикл заканчивается. 

### Пример (классика: вывести 1..5)
```py
i = 1
while i <= 5:
    print(i)
    i += 1
```
Идея: `i` увеличивается, и в какой-то момент условие `i <= 5` станет ложным.   

---

## 2) Частая ошибка: забыли изменить счётчик → бесконечный цикл
```py
i = 1
while i <= 3:
    print(i)
```
Здесь `i` не меняется, условие всегда True → будет печатать бесконечно.  

✅ Правильно:
```py
i = 1
while i <= 3:
    print(i)
    i += 1
```
Выведет: `1 2 3`. 

---

## 3) `break` — принудительно завершить цикл
`break` мгновенно прерывает цикл, даже если условие ещё True. 

Пример (остановка на 5):
```py
i = 1
while i <= 10:
    print(i)
    if i == 5:
        break
    i += 1
```


### `break` с пользовательским вводом
```py
while True:
    user_input = input("Введите 'exit', чтобы завершить цикл: ")
    if user_input == "exit":
        break
    print("Вы ввели:", user_input)
```


---

## 4) `continue` — пропустить текущую итерацию
`continue` пропускает оставшийся код **внутри текущей итерации** и переходит к следующей проверке условия.   

Пример (печатаем только нечётные числа):
```py
i = 0
while i < 5:
    i += 1
    if i % 2 == 0:
        continue
    print(i)
```
Вывод: `1 3 5`.   

Важно из урока: ставить `continue` **последней строкой** тела цикла бессмысленно (ничего не “пропускается”). 

---

## 5) Бесконечный цикл
**Бесконечный цикл** — цикл, который не завершится сам, потому что условие всегда True. Часто пишут так:
```py
while True:
    ...
```
Обычно его останавливают через `break` при нужном условии.  

Пример:
```py
while True:
    user_input = input("Введите 'stop', чтобы завершить цикл: ")
    if user_input == "stop":
        print("Цикл завершён.")
        break
```
 

---

## 6) Конструкция `while/else`
`else` выполнится **только если цикл завершился “нормально”**, то есть **без `break`**.  

### Пример
```py
i = 1
while i <= 5:
    print(i)
    i += 1
else:
    print("Цикл завершён без прерываний.")
```
 

### Важно: `continue` НЕ мешает `else`
`continue` пропускает часть итерации, но цикл всё равно может завершиться нормально → `else` выполнится.

## 7) Модуль `random`
`random` — стандартный модуль для генерации случайных чисел и выбора случайных элементов.   

### 7.1 `random.random()` → float от 0.0 до 1.0
```py
import random
print(random.random())
```

### 7.2 `random.randint(a, b)` → int от a до b (обе границы включительно)
```py
import random
print(random.randint(1, 10))
```

### 7.3 `random.randrange(start, stop, step)` → int (stop НЕ включается)
```py
import random
print(random.randrange(1, 10))      # 1..9
print(random.randrange(1, 10, 2))   # 1,3,5,7,9
```

## 8) Быстрые задания из урока (ответы и разбор)
### 8.1
```py
i = 1
while i <= 3:
    print(i)
```
Ответ: **бесконечный цикл** (нет изменения `i`).  

### 8.2
```py
i = 1
while i <= 3:
    print(i)
    i += 1
```
Ответ: **1, 2, 3**. 

### 8.3
```py
i = 10
while i > 7:
    i -= 1
print(i)
```
Ответ: **7** (цикл закончит, когда `i == 7`, потом `print(i)` вне цикла).   

### 8.4 (break)
```py
i = 1
while i <= 5:
    if i == 3:
        break
    print(i)
    i += 1
```
Ответ: **1, 2**.   

### 8.5 (continue)
```py
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
```
Ответ: **1, 2, 4, 5**.  

### 8.6 (`while/else`, continue не мешает)
```py
i = 1
while i <= 3:
    if i == 2:
        i += 1
        continue
    print(i)
    i += 1
else:
    print("Цикл завершён.")
```
Ответ: **1, 3, затем 'Цикл завершён.'** (нет `break`).  

### 8.7 (`while/else` не сработает при `break`)
```py
i = 1
while i <= 4:
    if i == 3:
        break
    print(i)
    i += 1
else:
    print("Цикл завершён.")
```
Ответ: **1, 2** (без строки `else`).

## 9) Практические задания (готовые решения)

### 9.1 Обратный отсчёт
Задача: вывести числа от введённого до 1 и затем “Отсчёт завершён!”.   
```py
num = int(input("Введите начальное число: "))
while num > 0:
    print(num)
    num -= 1
print("Отсчёт завершён!")
```


### 9.2 Сумма положительных чисел (до ввода 0)
Суммировать только положительные, при отрицательных — пропуск (`continue`), при `0` — выход (`break`). 
```py
total_sum = 0

while True:
    num = int(input("Введите число: "))

    if num < 0:
        continue

    if num == 0:
        break

    total_sum += num

print("Общая сумма положительных:", total_sum)
```

## 10) Домашка (шаблоны решений)

### 10.1 Сумма цифр числа
Идея: пока число не 0 — брать последнюю цифру через `% 10` и убирать её `// 10`. 
```py
n = int(input("Введите число: "))
n = abs(n)

s = 0
while n > 0:
    s += n % 10
    n //= 10

print("Сумма цифр:", s)
```


### 10.2 Палиндром (для целого числа)
Идея: развернуть число “в обратном порядке” и сравнить.   
```py
n = int(input("Введите число: "))
original = n
n = abs(n)

rev = 0
while n > 0:
    rev = rev * 10 + (n % 10)
    n //= 10

if abs(original) == rev:
    print(f"Число {original} является палиндромом.")
else:
    print(f"Число {original} не является палиндромом.")
```


### 10.3 “Проверь интуицию” — угадай число за 10 попыток
Требования: число 1..100, 10 попыток, подсказка “больше/меньше”, оценка по попыткам.   
```py
import random

secret = random.randint(1, 100)
attempts = 10
tries = 0

print("Угадайте число от 1 до 100. У вас 10 попыток.")

while tries < attempts:
    guess = int(input("Ваше предположение: "))
    tries += 1

    if guess == secret:
        print(f"Поздравляем! Вы угадали число: {secret}.")
        break
    elif guess < secret:
        print("Загаданное число больше вашего")
    else:
        print("Загаданное число меньше вашего")
else:
    print(f"Попытки закончились. Загаданное число было: {secret}")

# оценка (примерно как в примерах урока)
if guess == secret:
    if tries == 1:
        print("Отличная интуиция! Вы угадали с первой попытки!")
    elif tries <= 3:
        print(f"Вы угадали число за {tries} попытки. Отличный результат!")
    elif tries <= 6:
        print(f"Вы угадали число за {tries} попыток. Хороший результат!")
    else:
        print(f"Вы угадали число за {tries} попыток. Можно лучше :)")
```

## 11) Мини-шпаргалка
```text
while condition:
    ...

break    -> выйти из цикла сразу
continue -> пропустить итерацию

while/else:
else выполнится только если цикл закончился без break
continue не блокирует else

random:
random.random()       -> float 0..1
random.randint(a, b)  -> int a..b (включая b)
random.randrange(a,b,step) -> int, b не включается
```
