# Python Fundamentals — Урок 1: Основы работы с Python

Основа по теме: что такое Python, как он выполняет код, переменные, синтаксис, комментарии/PEP 8 и первые функции `print()` / `input()`.

## 1) Python: что это и почему его любят
**Python** — высокоуровневый язык программирования, известный простотой и читабельностью.

Ключевые особенности:
- **Простота** (код читается почти как английский текст)
- **Динамическая типизация** (тип определяется автоматически)
- **Интерпретируемость** (код выполняется “по строкам”)
- **Кроссплатформенность** (Windows/Linux/macOS)
- **Богатая стандартная библиотека**
- Поддержка **ООП** и **функционального подхода**
- Большое **сообщество** и экосистема библиотек

---

## 2) Интерпретатор vs компилятор (важная идея)
### Компилятор
- Сначала **переводит весь код** в машинный код
- Потом запускает
- Обычно быстрее на исполнении, но дольше подготовка

### Интерпретатор
- Выполняет код **построчно**, переводя “на лету”
- Удобно для обучения и отладки

✅ **Python — интерпретируемый язык**: запуск без предварительной компиляции.

---

## 3) Переменные и данные
### Что такое переменная
**Переменная** — именованная область памяти, где хранится значение, доступное по имени.

### Примеры данных
- `int` (целые): `5`, `-3`, `42`
- `float` (вещественные): `3.14`, `-0.001`, `2.0`
- `str` (строки): `"Привет"`, `'Мир'`

---

## 4) Оператор присваивания `=`
Оператор присваивания связывает переменную со значением.

```python
x = 10
name = "John"
```

⚠️ Не путать:
- `=` — присваивание
- `==` — сравнение (позже)

---

## 5) Правила именования переменных (чтобы не ловить ошибки)
Коротко и по делу:
- Имя может содержать **буквы, цифры, `_`**
- Начинается **только** с буквы или `_` (не с цифры)
- **Регистр важен**: `age` и `Age` — разные
- Нельзя использовать **ключевые слова** (`if`, `for`, `while`, …)
- Не называй переменные именами встроенных функций (`print`, `int`, `max`, …)
- По стилю: **snake_case** (`user_name`, `total_score`)
- Имя должно быть **понятным**, но не слишком длинным

✅ Хорошо:
```python
user_name = "Alice"
total_score = 100
_price = 10
```

❌ Плохо:
```python
2nd_place = 1       # начинается с цифры
my-variable = 10    # дефис нельзя
my variable = 10    # пробел нельзя
total%items = 5     # % нельзя
if = 42             # ключевое слово
```

## 6) Синтаксис Python: “закон отступов”
**Синтаксис** — правила написания кода, чтобы интерпретатор понял программу.

Главные правила:
- Обычно **1 строка = 1 инструкция**
- `;` можно, но **не рекомендуется**
- Блоки кода задаются **двоеточием `:`** и **отступами** (обычно 4 пробела)

Пример блока:
```python
a = 5
if a == 5:
    print("Hi")
    print("Hello")
```

### Ошибки синтаксиса (SyntaxError)
Частые причины:
- неправильные отступы
- забыли `:` после `if/for/while/def/...`
- ошибки в кавычках
- недопустимые символы в имени переменной

## 7) Комментарии
### Однострочный комментарий
```python
# Это комментарий
x = 5  # комментарий в конце строки
```

### Многострочный комментарий (или docstring-стиль)
```python
"""
Это многострочный комментарий/описание.
Может занимать несколько строк.
"""
```
(Также можно `''' ... '''`)

---

## 8) PEP 8 — базовые правила стиля
PEP 8 — рекомендации по оформлению кода, чтобы его было легче читать и поддерживать.

Минимум, который стоит соблюдать сразу:
- **Отступы:** 4 пробела
- **Длина строки:** до 79 символов
- **Имена:** `snake_case` для переменных/функций, `CamelCase` для классов, `UPPER_CASE` для констант
- Пробелы вокруг операторов: `x = 5`, `a + b`
- Импорты — в начале файла, группировать пустыми строками

---

## 9) Функции: что это и как вызвать
**Функция** — готовый блок кода, который выполняет задачу.

Чтобы вызвать функцию:
1) написать имя  
2) поставить `()`  
3) при необходимости — передать аргументы

```python
print("Привет, мир!")
```

---

## 10) `print()` — вывод на экран
```python
print("Hello, Python!")
print("Число:", 10)
print()  # выведет пустую строку
```

---

## 11) `input()` — ввод от пользователя
`input()` **всегда возвращает строку** (`str`).

```python
name = input("Введите ваше имя: ")
print("Привет, " + name + "!")
```

## 12) Мини-практика (как на занятии)
### 12.1 Приветствие
```python
print("Hello, Python!")
```

### 12.2 Ввод имени и вывод
```python
name = input("Введите ваше имя: ")
print(name)
```

### 12.3 Личное досье (имя + возраст + город)
```python
name = input("Введите имя: ")
age = input("Введите возраст: ")
city = input("Введите город: ")
print(f"{name} {age} лет, проживает в городе {city}")
```

### 12.4 Повторитель текста (3 раза)
```python
text = input("Введите текст: ")
print(text)
print(text)
print(text)
```

## 13) Быстрые ответы на задания (самопроверка)
- Оператор присваивания: **`=`**
- Корректные имена переменных из списка: **student_name, age_25, is_student, total_score**
- PEP 8 отступы: **4 пробела**
- Многострочный комментарий: **""" ... """** и **''' ... '''**
- `print(a b)` → **ошибка синтаксиса**
- “что делает вызов функции?” → **выполняет ранее написанную функцию**
- `print()` = вывод, `input()` = ввод
