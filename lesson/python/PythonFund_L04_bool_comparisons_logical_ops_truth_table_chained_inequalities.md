Конспект по теме: логический тип данных, truthy/falsey, операторы сравнения, `and/or/not`, приоритет, таблица истинности, двойные неравенства.

---
## 1) Логический тип данных `bool`
`bool` (*boolean*) — тип данных с **двумя** значениями:
- `True` (истина)
- `False` (ложь)

Логические значения помогают программам принимать решения (в условиях, циклах и т.д.). 
Прямое создание:
```py
is_finished = True
has_access = False
```

---

## 2) Приведение типов к `bool` (truthy / falsey)
В Python многие значения интерпретируются как `True` или `False` при преобразовании в `bool`.

### 2.1 Что считается True (truthy)
- любое **ненулевое** число (`42`, `-2`, `3.14`)
- любая **непустая** строка (`"0"`, `"Python"`, `"False"`)
- непустые коллекции (`[1]`, `{"a": 1}`, `(1,)`, …)

### 2.2 Что считается False (falsey)
- `None`
- `0`
- пустые последовательности/коллекции: `""`, `[]`, `{}` и т.п.

### 2.3 Явное преобразование: `bool()`
```py
print(bool(5))        # True
print(bool(0))        # False
print(bool(None))     # False
print(bool([]))       # False
print(bool("Hello"))  # True
```

---

## 3) Операторы сравнения
Операторы сравнения в Python возвращают **всегда** `True` или `False`.

| Оператор | Смысл |
|---|---|
| `==` | равно |
| `!=` | не равно |
| `>`  | больше |
| `<`  | меньше |
| `>=` | больше или равно |
| `<=` | меньше или равно |

Примеры:
```py
a = 10
b = 5
print(a == b)  # False
print(a != b)  # True
print(a > b)   # True
print(a < b)   # False
print(a >= 10) # True
print(b <= 4)  # False
```

---

## 4) Логические операторы: `and`, `or`, `not`
Логические операторы комбинируют условия и возвращают `True/False`.

| Оператор | Смысл |
|---|---|
| `and` | логическое «И» (True, если оба условия True) |
| `or`  | логическое «ИЛИ» (True, если хотя бы одно True) |
| `not` | логическое «НЕ» (инверсия) |

Примеры:
```py
print(True and False)  # False
print(True or False)   # True
print(not True)        # False
```

---

## 5) Комбинирование условий
Можно писать сложные выражения, комбинируя `and/or/not`.

Пример из материалов:
```py
a = 5
b = 10
print(a > 0 and b > 0 or a == 5)
```
Идея: результат будет `True`, потому что оба числа > 0 (часть с `and` истинна), и также `a == 5` истинно. 

---

## 6) Приоритет логических операторов
Порядок выполнения:
1) `not`  
2) `and`  
3) `or`

Пример разбора (из материалов):
```py
a = True
b = True
c = True
print(not a or b and c)
```
Сначала `not a`, потом `b and c`, затем `or`.

### Скобки меняют приоритет
```py
result = not (a or b) and c
```
Теперь сначала считается `(a or b)`, затем `not`, затем `and`. 

---

## 7) Приоритет математики и сравнения
Математические операции выполняются **раньше**, чем сравнения.

```py
result = 3 + 2 > 4
print(result)  # True, потому что сначала 3+2=5, затем 5 > 4
```

---

## 8) Таблица истинности (Truth Table)
Таблица истинности показывает результаты `and/or/not` для всех комбинаций.

| A     | B     | A and B|A or B| not A |
|-------|-------|-------|-------|-------|
| True  | True  | True  | True  | False |
| True  | False | False | True  | False |
| False | True  | False | True  | True  |
| False | False | False | False | True  |

---

## 9) “Альтернативная интерпретация” логических операторов
В материалах логические операции сопоставляются с арифметикой:
- `and` ≈ умножение (`*`)
- `or`  ≈ сложение (`+`)
- `not` ≈ инверсия

(Это именно **аналогия** для понимания логики.)

---

## 10) Двойные неравенства (chained comparisons)
В Python можно писать проверки диапазона “цепочкой”:

```py
min_value < x <= max_value
```

Пример:
```py
x = 5
print(1 < x <= 10)
```

Эквивалентно:
```py
print(1 < x and x <= 10)
```

Двойные неравенства читаются проще и короче.

## 11) Практическая работа (из урока) — решения

### 11.1 Число в диапазоне
Задача: получить два числа и вывести, попадает ли первое число в диапазон от 1 до второго (включая границы). 

Решение:
```py
x = int(input("Введите первое число: "))
y = int(input("Введите второе число: "))

print(1 <= x <= y)
```

### 11.2 Сравнение чисел
Задача: получить 3 числа и сравнить первое со вторым и третьим (больше/меньше/равно). 
Решение:
```py
a = int(input("Введите первое число: "))
b = int(input("Введите второе число: "))
c = int(input("Введите третье число: "))

print("Первое больше второго:", a > b)
print("Первое меньше третьего:", a < c)
print("Первое равно третьему:", a == c)
```

## 12) Домашнее задание — подсказки решений

### 12.1 Логические операции + сравнения (and/or/not + ==/!=)
Задача: получить два логических значения и вывести результаты `and`, `or`, `not` (для первого), а также `==` и `!=`. 

⚠️ Ввод из `input()` — это строка. Один простой способ:
- пользователь вводит `True` или `False`.

```py
v1 = input("Enter first value (True/False): ").strip().lower() == "true"
v2 = input("Enter second value (True/False): ").strip().lower() == "true"

print("and:", v1 and v2)
print("or:", v1 or v2)
print("not:", not v1)
print("equal:", v1 == v2)
print("not equal:", v1 != v2)
```

### 12.2 Проверка условий (свет / окно)
Задача: два логических значения → проверить:
- оба ли условия выполнены
- хотя бы одно выполнено 

```py
light_on = input("Свет включён? True/False: ").strip().lower() == "true"
window_open = input("Окно открыто? True/False: ").strip().lower() == "true"

print("Оба условия выполнены?", light_on and window_open)
print("Хотя бы одно условие выполнено?", light_on or window_open)
```

### 12.3 (*) Стоимость мобильного тарифа
Условия: базовая стоимость 30€, всё сверх 500 МБ — по 0.2€ за МБ. 

```py
base = 30
limit_mb = 500
over_price = 0.2

mb = int(input("Введите использованные мегабайты: "))

over_mb = mb - limit_mb
if over_mb < 0:
    over_mb = 0

overpay = over_mb * over_price
total = base + overpay

print("Переплата:", overpay)
print("Общая стоимость:", total)
```

## 13) Мини-шпаргалка
```text
bool: True / False

Falsey: 0, None, "", [], {}, ...
Truthy: всё остальное (ненулевое, непустое)

Сравнение: == != > < >= <=

Логика: and / or / not
Приоритет: not -> and -> or
Скобки меняют порядок

Диапазон: 1 <= x <= 10  (вместо 1 <= x and x <= 10)
```
