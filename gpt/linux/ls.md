# ls — шпаргалка

Коротко: `ls` выводит содержимое директорий; комбинации опций дают подробные списки, сортировки и фильтры.
#ls, #linux, #filesystem
---

## Синтаксис

```
ls [OPTIONS] [FILE...]
ls                         # по алфавиту
ls /var/log                # конкретная директория
ls file.txt                # один файл
```

## Ключевые опции

- `-l` — длинный формат (права, владелец, размер, время)
- `-a` — показать скрытые (`.` и `..`)
- `-A` — как `-a`, но без `.`/`..`
- `-h` — размеры в человекочитаемых единицах (`1K`, `4M`)
- `-R` — рекурсивно
- `-t` — сортировать по времени изменения (новые сверху)
- `-c` / `-u` — сортировать по времени изменения/доступа и показывать вместо `mtime`
- `-S` — сортировать по размеру
- `-r` — обратный порядок
- `-1` — один файл в строке (удобно для скриптов)
- `--color=auto` — раскрашивать по типу (включено в большинстве дистрибутивов)
- `-d` — показывать только сами каталоги, не их содержимое
- `-X` — сортировать по расширению
- `-F` — добавлять символы (`/`, `*`, `@`) для различения типов

## Форматы вывода

- `ls -l` — таблица с правами, владельцем, размером, датой
- `ls -lh` — тот же формат, но `-h` для удобоваримого размера
- `ls -ld */` — показать только каталоги в текущей директории
- `ls -ltr` — посмотреть последние изменённые файлы внизу
- `ls -aR` — рекурсивно показать всё, включая скрытые
- `ls -1` — удобно передавать в другие команды (`ls -1 | grep pattern`)

## Советы и хитрости

- Используйте `alias ll='ls -alF'` и `alias la='ls -A'` в `~/.bashrc`
- Для проверки размера каталогов используйте `du -sh *` (так как `ls` показывает только размеры файлов)
- Комбинируйте `ls` с `grep` (`ls | grep pattern`), чтобы быстро фильтровать
- `ls -d */` полезно, когда надо только список директорий
- `ls --group-directories-first` — каталог выводится первым (если поддерживается)
- `ls` без аргументов использует `COLUMNS` и выводит аккуратно; при нестандартной ширине `export COLUMNS=80`
