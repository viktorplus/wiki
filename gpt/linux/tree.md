# tree — шпаргалка

Коротко: `tree` выводит древовидную структуру каталогов/файлов, удобно для визуального обзора структуры и быстрого счёта содержимого.
#tree, #linux, #filesystem
---

## Синтаксис

```
tree [OPTIONS] [DIRECTORY]
tree .          # дерево текущего каталога
tree /var/www  # структура web-проекта
tree -L 2       # глубина 2 уровня
```

Если `DIRECTORY` не указан, используется текущий каталог. `tree` печатает диаграмму с символами ветвлений и подчитает файлы/каталоги.

## Полезные опции

- `-L N` — ограничить глубину (N уровней)
- `-d` — показывать только каталоги, без файлов
- `-a` — включить скрытые файлы и каталоги (`.*`)
- `-H base` — HTML-вывод (для генерации документации)
- `-f` — отображать полные пути (для скриптов)
- `-i` — не выводить ASCII-арт ветвления (для parse)
- `-F` — добавить символ (/, *, =) в конце имени (как `ls -F`)
- `-h` — человекочитаемые размеры файлов (например, `4K`, `1M`)
- `--du` — показать суммарный размер каталогов (`du`-подобно)
- `-P pattern` и `-I pattern` — включать или исключать по шаблону (wildcards)
- `-C` — цветной вывод (по умолчанию при поддержке терминала)
- `-o file` — записать вывод в файл
- `--charset=CHARSET` — задать кодировку (используется в Windows/старых терминалах)
- `--noreport` — не печатать в конце количество файлов/каталогов

## Примеры

- Показать только каталоги в проекте:
  ```
  tree -d -L 3
  ```
- Вывести дерево со скрытыми файлами:
  ```
  tree -a
  ```
- Список файлов с полными путями:
  ```
  tree -f -I '*.log'
  ```
- HTML-отчёт для документации (используется вместе с `-H` и `--charset`):
  ```
  tree -H . -L 2 -o tree.html
  ```
- Получить размеры каталогов:
  ```
  tree --du -h -L 1
  ```
- Напечатать структуру, исключив исходники компиляции (`build`, `.git`):
  ```
  tree -I 'build|.git'
  ```
- Быстрая проверка, есть ли `.env` в дереве:
  ```
  tree -P '.env' -I '*'
  ```

## Советы

- Используйте `tree -L` вместо `ls -R`, когда нужна краткая структура.
- Для больших деревьев добавляйте `| head`, `| less` или записывайте в файл (`tree -L 2 > structure.txt`).
- Если `tree` отсутствует, установите через пакетный менеджер (`sudo apt install tree`, `brew install tree`).
- Сочетайте с `grep` (перед `tree` через `-P`) или `fd` для фильтрации по шаблонам.
